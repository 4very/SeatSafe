How to use Seat Safe

Seat Safe is a seat reservation app that provides an intuitive interface for hosts to make a reservation chart and for users to select what spots they would like to select for the event.

View List of Events

-Go to “Browse Events” page via the navigation bar at the top
-From here, you will see a list of all of the public events

Creating an event

-Go to the “Create Event” section of website through the button on top right of the navigation bar
-Enter in all relevant information for the event which includes the event’s name, your contact email, and an image for the seating chart of the event (you must provide a link to your own seating chart)
-Chose whether you want the event to be public or private
-Add/remove seating groups by including a name and number of seats and then clicking the 'Add Group button'
-When you are done select 'Create Event' at the bottom to publish your event
-You will be given a link for your event details. 
Keep track of this link, as it is the only way to view and cancel your event 

Make a Reservation

-On the “Browse Events” page, select “View” on an event to view that events information
-Click on "Create Reservation" to create a reservation for that event
-Enter your name and email into the text fields
-Select how many seats you want in a particular group
-Hit “Reserve” to submit your reservation
-You will be given a link for your reservation details.
Keep track of this link, as it is the only way to view and cancel your reservation 

Cancel a Reservation

-Navigate to the private link you received when you made your reservation
-Click the red 'Cancel Reservation' at the bottom of the page

Cancel an Event

-Navigate to the private link you received when you made your event
-Click the 'Cancel Event' button at the bottom of the page
